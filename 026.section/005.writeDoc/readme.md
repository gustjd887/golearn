# 패키지, 메서드, 함수 위에 주석으로 작성
* 대문자로 시작, 마침표로 마무리
* 여러 줄 작성은 바로 밑줄에 주석으로 이어감
* 주석 사이 공백 행이 있으면 안됨
* export되지 않는 소문자로 시작하는 요소는 godoc에 들어가지 않음. 그래도 주석은 작성해두는게 좋다고 함.
* example은 test에서 정의하는 것 같음.
* 주석은 K(대문자)eword Identifier_name is ... 의 형식으로 작성해야 함.
